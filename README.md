# attractors

![attractors](img/attractors.jpg)

```
Controls:
   wasd --- move camera
   mouse -- aim camera
   r ------ reset particle positions
   p,o ---- pause,resume evolution
   l ------ manually advance evolution
   z,x ---- scale attractor smaller,larger
   arrows - rotate attractor
   1 ------ Lorenz attractor
   2 ------ Roessler attractor
   3 ------ Lu Chen attractor
```
